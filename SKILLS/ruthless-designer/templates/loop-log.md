# Loop Log

Use one block per material pass.

```text
Loop N:
Source item:
Before evidence:
Action:
Files/artifacts changed:
After evidence:
Verdict: substantially better | mixed | flat | worse
Remaining concern:
Next action:
```

## Direction Reset

Use when two consecutive verdicts are `flat` or `worse`.

```text
Direction failed because:
Kill:
Keep:
New direction:
New signature move:
Proof target:
```

# Skills

This project ships one Codex skill:

- [`ruthless-designer`](./ruthless-designer/SKILL.md): distinctive greenfield UI creation, broad redesign, aggressive evidence-backed critique, and visual proof.

Use it when an interface must be invented, meaningfully re-composed, or subjected to a design autopsy. Use a narrower workflow for isolated component fixes, accessibility/performance-only repairs, or code-only review.
